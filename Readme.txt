Follow these steps:

Open the config.properties file located in the config directory.

Set the value of the browser property to either chrome or firefox.

Example:

browser=chrome

Running Tests
To run the test suite, follow these steps:

Make sure you have Java and Maven installed on your system.

Clone the repository to your local machine.

Navigate to the project directory.

Configure the config.properties file as described above.

Open a terminal or command prompt.

Run the following Maven command to execute the tests:

mvn clean test

Browser Versions:
Make sure you have the following browser versions installed on your system:

Google Chrome: Version 114.0.0
Mozilla Firefox: Version 114.0.0
